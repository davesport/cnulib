This project combines the information & education of libraries with the engaging, interactive design of shopping websites. The goal is to embed interactive elements into this webpage, so that users exercise their information literacy skills by just going onto this site. 

Features include: intra-links for different research steps, database filters, database hover descriptions, drop-down menus. 


This webpage was made using a Bootstrap template.
